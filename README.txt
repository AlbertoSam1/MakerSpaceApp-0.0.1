Request ID Generation
- Totals to 14 characters
    - 4 Capital letters
    - 8 Numbers corresponding to current date
    - 2 Random with no seed selected

    Example: NAVD0111202145

Group ID Generation
- Totals to 8 Characters
    - Same 4 letters as Request ID
    - 4 Random numbers with no seed selected

Use current ones in send update request
